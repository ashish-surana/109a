Restaurant Recommender

Ashish Surana · Roman Burdakov · Ying Tian Wang · CSCI E-109A Yelp Data Challenge

Can we predict what star rating a user will give to a restaurant?
Knowing this information would be very usefull to provide all sorts of recomendations for a registered user. Our goal would be to mimic existing yelp where a registered user looks for restaurant recomendations within a certain area (zip code). Yelp, isn't just listing all places filtered by highest ratings, distance or number of reviews. Instead it applies its trained model to find places (excluding paid accounts that are boosted to the top), which most likely will receive good rating and review if a given user would visit it.

Exploratory Data Analysis

From our real life experience, the most important factors needed for recommendations are ratings given by user, average rating of a business, the number of reviews of a business and the number of reviews given by user. We also want to find the rating and number of review distribution given by state.


Proposed Questions

  1. How many stars a business usually get?
  2. How many reviews a business usually get?
  3. How many reviews a user usually give?
  4. How does the reviews_count and star of a business related?
  5. what is the rating distribution given by state?
  6. what is the review distribution given by state?


Challenges

The challenge is that the dataset is very big, and there are many categories and attributes to sort out.

How many stars a business usually get?
Answer: Most businesses have starts of 4 and 3.5. A small number of business has start that is under 1.5

How many reviews a business usually get?
Answer: We can see most of the business have reviews that is less than 10, very few business have more than 300 reviews.

How many reviews a user usually give?
Answer: We can see the majority of the user give less than 10 reviews to businesses

How does the reviews_count and star of a business related?
Answer: Business that has more than 1000 reviews are most likey to get 2.5 to 4 stars.


What's the star rating distribution by states?
Answer: Most of the states have ratings between 4 to 5.

What's the review count distribution by states?
Answer: Arizona has the highest number of reviews.

Baseline

The baseline linear regression model is created by OLS api in statsmodels, which minimize the sum of the squares of the differences between the observed responses. The resulting baseline rating can be expressed by a simple linear combination of single-user bias across all restaurants and specific restaurant bias for all users. The R2_score is really low (<0.36) here, which indicating the model is not working well to recommend user restaurants. However, according to the reference for Netflix Prize, modeling these biases turned out to be fairly important.

Ensemble using Random Forest¶

A random forest is a meta estimator that fits a number of classifying decision trees on various sub-samples of the dataset and use averaging to improve the predictive accuracy and control over-fitting. The sub-sample size is always the same as the original input sample size but the samples are drawn with replacement.

def ensemble(Xtrain, ytrain):
#     n_estimators is the number of trees in the forest.
# The number of features to consider when looking for the best split. 
# Here max_features is a percentage and int(max_features * n_features) features are considered at each split.
#     param_dict = OrderedDict(
#         n_estimators = [50, 200, 300],
#         max_features = [0.2, 0.4, .6]
#     )

    param_dict = OrderedDict(
        n_estimators = [50, 100, 200, 300],
        max_features = [0.2, 0.4]
    )

    start_timer()
    results = {}
    estimators= {}
    for n, f in product(*param_dict.values()):
        params = (n, f)
        print(f'Creating RandomForestRegressor using {n} estimators and {f}% max features: ')
        est = RandomForestRegressor(oob_score=True, 
                                    n_estimators=n, max_features=f, n_jobs=-1)
        est.fit(Xtrain, ytrain)
        results[params] = est.oob_score_
        print('OOB score is:', est.oob_score_)
        estimators[params] = est
    print(results)
    outparams = max(results, key = results.get)
    stop_timer()
    print(outparams)
    
    # get the regressor corresponding to the outparams
    rf1 = estimators[outparams]
    return (rf1,results)
